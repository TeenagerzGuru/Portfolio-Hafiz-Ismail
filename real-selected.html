KHAWARIZMI LIBRARY

The description:

Khawarizmi Library System is a modern and user-friendly library management solution designed to efficiently organize, track, and manage books and users at UNITAR.

The system allows librarians to easily catalog books, manage borrow and return transactions, and monitor overdue items.

Users can browse the digital catalog, search for books by title, author, or genre, and check availability in real-time.

TRAFFIC CHAOS

The description:

Traffic Chaos is an endless highway game to avoid traffic.

Player will face three type of terrain which is: Desert, Antarctica and Summer.

To control by using W - forward, S - slow down, A - left turn, D - right turn. Space - Handbrake

Automatically update the highscore

Be alert and avoid the traffic!
