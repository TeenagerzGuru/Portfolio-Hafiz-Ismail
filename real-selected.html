The description:

GOMOKU GAME

The description:

A classic board game for two players where the objective is to be the first to get five of your pieces in a row.

Players alternate placing X and O markers on the board. The first player to achieve five consecutive pieces horizontally, vertically, or diagonally wins.

Developed using HTML, CSS, and JavaScript with intelligent game logic and responsive design.
